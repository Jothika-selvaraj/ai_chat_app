# 🚀 AI Flow App (MERN Stack + React Flow + OpenRouter)

## 📌 Overview

This project is a full-stack MERN application that demonstrates integration of frontend, backend, AI API, and database.

The application allows users to:
- Enter a prompt in an input node
- Click "Run Flow" to generate AI response
- View the result in a connected node
- Save the prompt and response to MongoDB

---

## 🧠 Tech Stack

### Frontend
- React.js
- React Flow

### Backend
- Node.js
- Express.js

### Database
- MongoDB 
### AI Integration
- OpenRouter API  
- Model: `mistralai/mistral-7b-instruct:free`

### Deployment
- Frontend: Vercel  
- Backend: Render  

---

## ✨ Features

- Interactive node-based UI using React Flow  
- AI response generation via OpenRouter  
- Secure backend API (no API keys exposed)  
- MongoDB database integration  
- Save prompt & response functionality  
- Fully deployed application
